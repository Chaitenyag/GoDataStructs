# Introduction
This is a project to recreate the popular ADT's in Golang

## ADT's Implemented:
- Fixed List
- Dynamic List
- Linked List
- Doubly Linked List
- BST
- Stack
- Queue
- Deque
- Steque
- AVL Trees

## ADT's In-Progress:
- Graph

- Red Black Trees

- B Trees
- Trie
- Binary Heap
- Priority Queue
- Indexed Priority Queue
